# Development

## Directory Struture
### .github :
- Templates for issues and pull requests, along with the workflow for deploying the project to the server.

### Meetings : 
- A directory with summaries of the meetings the group had regarding the project's development.

### Reports :

- Documentation of the project divided into its different iterations, final report and diagrams of the project.

### Milestones

- A directory containing PowerPoint used in presentations .

## Branchs

- `main`:This branch contains the latest stable version of the project and reports.

- `dev`:This branch contains the most recent version of the project.

### Branch structure

- ```<label>/<Developmentcomponent>/<IssueRelatedToTheComponent>```
